# 🚁 Premium Drone Landing Page

A product landing page built with HTML5 and CSS3 for showcasing a drone and its specifications.

## 🎥 Demo



https://github.com/user-attachments/assets/a8fc3f2b-583d-40d2-af02-462799c5b049



## 🚀 Features

* Clean landing page layout.
* Product specifications section.
* Product comparison table.
* Flexbox-based layouts.
* CSS positioning.
* Hover effects.

## 🛠️ Technologies Used

* HTML5
* CSS3
* Flexbox
* CSS Positioning

## 🧠 Challenges & Learnings

The main challenge in this project was building and styling the product comparison table while maintaining a clean layout using Flexbox and CSS positioning.
